# Student Management System

A simple **Spring Boot CRUD web application** for managing student records.

This project allows users to add, edit, delete, and search student information through a clean dashboard interface.

---

## Features

- Add new students
- Edit student information
- Delete students
- Search students by name
- Form validation
- Prevent duplicate Student ID
- Dashboard UI
- Responsive Bootstrap design

---

## Tech Stack

Backend:
- Java
- Spring Boot
- Spring Data JPA

Frontend:
- Thymeleaf
- Bootstrap 5
- HTML / CSS

Database:
- MySQL

Build Tool:
- Maven
